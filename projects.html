Projects

Convert to sound

Convert to Sound is a C++ application that converts user-inputted text into spoken audio. The program leverages the eSpeak-ng speech synthesis engine for text-to-speech conversion and uses the SDL2 library to handle audio playback. After entering text, the program synthesizes speech and plays the resulting audio file in real time. This project demonstrates the integration of system calls, external libraries, and basic audio handling in C++.

Snake game

This is a classic Snake Game built entirely with HTML5 Canvas and vanilla JavaScript, without any frameworks. The game features smooth animations, custom visuals (snake GIF and apple image), and an interactive "Start Game" button. It's a fun demonstration of working with canvas-based drawing, animation loops, keyboard controls, and mouse interactions.

Crazy 8

Crazy 8 is a digital card game built with React as a part of a collaborative school project. The application simulates the classic Crazy Eights card game and was developed with a focus on teamwork, clean component structure, and responsive UI.
